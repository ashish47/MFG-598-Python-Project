# MFG-598-Python-Project
**Installation**
- pip3 install flask

**Running**
- Copy 'templates' folder and 'engine.py' to your local machine
- Run _**python3 engine.py**_
- Open a browser and type http://localhost:5000/ to access the UI

**Notes**
